# Custom Tab Expert

This knowledge base covers custom tab creation and configuration.

## Tab types

- **Lightning component tab**: Uses a Lightning web or Aura component.
- **Visualforce tab**: Embeds a Visualforce page.
- **Web tab**: Links to an external URL (e.g. help or internal site).
- **Object tab**: Exposes a custom object’s list/detail views.

## Creating a tab

1. Setup → Tabs → New (under Lightning Page Tabs or similar).
2. Choose tab type (Lightning, Visualforce, Web, Object).
3. Set label, name, and optional icon/style.
4. Add the tab to the desired app(s) in the app’s navigation items.
