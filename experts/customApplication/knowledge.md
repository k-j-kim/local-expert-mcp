# Custom Application Expert

This knowledge base covers custom application setup, configuration, and best practices.

## Key concepts

- **Application scope**: Define which objects and tabs your app uses.
- **Permissions**: Use permission sets or profiles to control access.
- **Tabs**: Custom tabs can link to external URLs or Visualforce pages.

## Common tasks

1. Create a custom app from Setup → App Manager → New Lightning App.
2. Add items (tabs, objects) to the app navigation.
3. Assign the app to profiles or permission sets.
